## 📋 SQL Queries Practiced (65 Queries)

During Day 1, I practiced a total of **65 SQL queries** covering different fundamental concepts.

### 🔹 Categories Covered

#### 1. Basic Retrieval Queries

* Selecting all records from tables
* Selecting specific columns

#### 2. Filtering Data

* Using `WHERE` clause
* Applying conditions with operators (`=`, `>`, `<`, `!=`)
* Using `AND`, `OR`, `NOT`

#### 3. Sorting Data

* Using `ORDER BY` (Ascending & Descending)

#### 4. Aggregate Functions

* `COUNT()`
* `SUM()`
* `AVG()`
* `MIN()`
* `MAX()`

#### 5. Grouping Data

* Using `GROUP BY`
* Filtering groups using `HAVING`

#### 6. String and Pattern Matching

* Using `LIKE`
* Wildcards (`%`, `_`)

#### 7. Handling NULL Values

* `IS NULL`
* `IS NOT NULL`
